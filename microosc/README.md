# MicroOsc

MicroOsc est une bibliothèque Open Sound Control (OSC) simple et légère pour Arduino, compatible avec les plateformes Arduino, Teensy, esp8266 et ESP32. MicroOsc peut envoyer et recevoir des messages OSC via Serial (SLIP) ou via UDP (Ethernet ou WiFi).

Code source et documentation complète (en anglais) : [MicroOsc](https://github.com/thomasfredericks/MicroOsc)
