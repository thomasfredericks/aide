# Guide de normes graphiques

Ce document permet de :
- Présenter le logo et son utilisation.
- Déterminer les deux couleurs principales et des sous couleurs. Ces couleurs doivent être utilisées en plus grande proportion à travers les supports visuels. 
- Établir et nommer la police principale qui est utilisée pour le logo et les titres dans les documents.
- Établir et nommer la police secondaire qui est utilisée pour le reste du texte.
