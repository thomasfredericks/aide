# Scan de réseau : Open-IP-Scanner

Open-IP-Scanner est un utilitaire open-source léger dédié à l'exploration et au diagnostic des réseaux locaux (LAN).

### Objectifs et fonctionnalités
* **Scanner des plages d'adresses IP** : identifier les hôtes actifs sur un segment réseau défini.
* **Récupérer les informations matérielles** : extraire les adresses MAC et tenter de résoudre les noms d'hôtes (hostnames).
* **Identifier les constructeurs** : analyser les préfixes des adresses MAC pour déterminer la marque des équipements connectés.
* **Simplifier l'administration** : localiser rapidement des périphériques sans interface (imprimantes, serveurs, objets connectés).

### Cas d'usage types
* **Vérifier la sécurité du réseau** : détecter d'éventuels intrus sur une connexion Wi-Fi.
* **Inventorier un parc informatique** : lister l'intégralité des terminaux actifs à un instant T.
* **Faciliter le dépannage** : confirmer la connectivité réseau d'un nouvel appareil.
