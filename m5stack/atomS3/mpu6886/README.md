# AtomS3 : accéléromètre et de gyroscope (mpu6886)

Le module AtomS3 **avec** écran est équipé d'un **capteur inertiel 6 axes (MPU6886)** qui permet de mesurer les mouvements et l’orientation. Lien officiel vers la page du produit : [AtomS3](https://docs.m5stack.com/en/core/AtomS3) 

Voici un exemple qui démontre son utilisation : [thomasofredericks/atoms3_accel_gyro - Codeberg.org](https://codeberg.org/thomasofredericks/atoms3_accel_gyro)
