# Maquettage

Le maquettage permet de déterminer si un projet est réalisable sur les plans technique, économique et opérationnel. Cela aide à identifier les obstacles potentiels et à planifier les ressources nécessaires. Dans le contexte des technologies interactives, où les innovations sont rapides, cette démarche devient encore plus critique.

Cette étape est composée de deux sous-étapes :
- [Scénarisation](./scenarisation/)
- [L'aspect technique (planification)](./technique/)
- Démo (faisabilité)
