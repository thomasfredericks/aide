# Expériences

Nous pouvons établir trois pôles de l'expérience :
- le jeu
- le cinéma
- l'art plastique



### Le jeu

Un jeu consiste à rendre une action suffisamment difficile pour stimuler, mais pas au point qu’elle devienne impossible — juste assez exigeante pour nécessiter le développement de compétences.
Un jeu est un système dans lequel on agit. Le jeu crée un défi. Il nécessite un engagement actif. Il développe des compétences.

- L'utilisateur prend des décisions.
- Le système réagit.
- L'utilisateur peut échouer.
- L'utilisateur peut s'améliorer.

### Le cinéma

Le cinéma consiste à organiser des images et des sons dans le temps afin de produire une expérience guidée — suffisamment structurée pour orienter l’émotion et la compréhension, sans laisser au spectateur le contrôle du déroulement. Le cinéma est un art dans lequel on observe. Il crée une immersion narrative. Il nécessite une attention soutenue. Il façonne une expérience émotionnelle et temporelle.

- L'utilisateur observe.
- Il ressent des émotions guidées.
- L'expérience se termine dans un temps défini.

### L'art plastique

L’art plastique consiste à donner forme à une présence dans l’espace — suffisamment construite pour orienter le regard, mais assez ouverte pour laisser place à l’interprétation. L’art plastique est un art que l’on contemple. Il crée une expérience sensible. Il ne prescrit pas de durée. Il invite à une exploration libre du sens.

- L'utilisateur rentre dans un espace.
- L'utilisateur contemple à sa guise.
- L'œuvre invite à l'interprétation.

## L'expérience d'un projet interactif

- Le projet, en tant que tel,ou son concept n’est pas l’expérience. Mais il permet de la créer.​
- Le projet est un vecteur : il génère une expérience dès lors qu’une personne entre en interaction avec lui.​
- L'expérience est une chose intangible et changeante.
- Pour le concepteur, la qualité de l’expérience est une préoccupation essentielle.​
- C’est sur la qualité de l’expérience que l’on juge les concepteurs.​  

Source : *p45 de L’art du Game Design de Jesse Schell, 2020.​*

- L'interaction transforme des lignes de code en art.

## Qualités d’une bonne expérience interactive

### **Clarté**  
L’interface est compréhensible, lisible et explicite.

### **Intuitivité**  
L’utilisateur sait quoi faire sans réfléchir.

### **Réactivité**  
Les retours sont rapides, visibles et rassurants.

### **Cohérence**  
L’ensemble est homogène, stable et prévisible.

### **Agentivité**  
L’utilisateur garde la main, peut agir ou revenir facilement.

### **Fiabilité**  
Elle fonctionne sans bug, sur tous les supports, pour tous les publics.
