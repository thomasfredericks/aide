# Tunnel réseau ngrok

ngrok est un outil de *tunelling* qui permettant d’exposer un serveur local à Internet via une URL publique sécurisée. Il permet de :
- Tester des *webhooks*, de partager une application en développement ou de simuler des interactions externes sans déploiement. 
- Créer un tunnel HTTPS redirigeant le trafic externe vers l’environnement local. 

Par exemple, il suffit de lancer `ngrok http 3000` pour exposer un serveur local accessible ensuite via une URL publique générée qui ressemble à `https://blubber-essence-zorg.ngrok-free.dev`.

Voir : [ngrok.com](https://ngrok.com).
